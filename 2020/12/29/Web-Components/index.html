Shadow DOM

It is a separate private DOM from the main one
but it is attached to one of the elements on the main one. –> it as a hidden and inaccessible part of the regular DOM.

  1. Shadow host: The DOM node of the main DOM that the shadow DOM is attached to.
  2. Shadow boundary: The line where the shadow DOM ends and the main DOM begins.
  3. Shadow root: The shadow tree’s root node.

Code basics

  1. open attachment: the shadow DOM can be accessed using JavaScript written on the main page.

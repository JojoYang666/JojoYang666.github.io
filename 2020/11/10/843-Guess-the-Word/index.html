843. Guess the Word

2020-11-10 Leetcode PV:
  1. everytime we update the possible list, in the matches list to narrow the range
  1. 将word在wordlist里面进行两两比较， 计算有多少个字符是相等的 –> list of list
    • example: worldist[a, b, c]
    • return [[1, 0, 0], [1, 0, 0], [1, 0, 0]] –> each item at idx j in list[i] represent how many chars are same for word i and j

–> 基本思想排除法， 缩小范围

  • 先随便guess 一个word a, 得到match b
  • 继续guess, 但是我们在与word a有相同match b的word中进行查找， 排除那些不同的， narrow范围
  1. call solve function:
    • possible: all the possible words
    • path: the guess we have made so far

300. Longest Increasing Subsequence

2020-11-11 Leetcode PV:

O(n*n) –> return max dp

O(n*logn) –> return length of dp

  1. dpdp array does not result in longest increasing subsequence, but length of dpdp array will give you length of LIS.
    Example:
    input: [0, 8, 4, 12, 2]
    dp: [0]
    dp: [0, 8]
    dp: [0, 4]
    dp: [0, 4, 12]
    dp: [0 , 2, 12

741. Cherry Pickup

2020-11-09 Leetcode PV:

  1. Person moves from [0, 0] to [N-1, N-1], then moves back from [N-1, N-1] to [0,0]

  2. Then we can convert the problem to 2 person moves from [0,0] to [N-1, N-1] at the same time

    • p1, p2 cannot arrive at the pos with the cherry at same time
    • grid[r1][c1] + (c1 != c2) * grid[r2][c2]

  3. Notice, after t steps, each position, we will have this property: r + c = t

  4. Thus, p1 -> (r1, c1). p2 -> (r2, c2)

    • r1 + c1 = r2 + c2
    • r2 = r1 + c1 - c2 (using this we can convert n^4 to n^3)

  5. Then when move, it has 4 possible situations

    • p1 down, p2 down: dp[r1+1][c1][c2]
    • p1 right, p2 down: dp[r1][c1+1][c2]
    • p1 down, p2 right: dp[r1+1][c1][c2+1]
    • p1 right, p2 right: dp[r1][c1+1][c2+1]

GIT

2020-06-22 Toolscheatsheet PV:

Stash

  • Git stash:
    • Hide current work
  • Git stash list:
    • View current stash
  • Git stash apply & git stash drop
    • Apply and delete
  • Git stash pop
    • Same as above
  • Git stash apply stash@{0}
    • Select stash
  • Git stash show []: 展示存储单元和最新存储的结果
    • Git stash show -p []: stash detail

Cherry-pick

  • git cherry-pick -n

Diff

  • git diff –staged
  • git diff –cached

REWRITING GIT HISTORY

  • git commit –amend: Replace the last commit with the staged changes and last commit combined. Use with nothing staged to edit the last commit’s message
  • git commit –amend -m “New commit message”:
    • what the command does is overwriting the most recent commit with the new one
    • `-m’ option allows you to write the new message on the command line without opening an editor session
  • git rebase -i —> 前闭后开

Rename

git branch -m

Create New Branch

git checkout -f -B local_TOT remotes/origin/TOT

git checkout

  1. git checkout -- file
    • git revoke the file in the working space, let it roll back to the latest status git commit/git add
  2. git checkout <other branch> -- <file path1> <file path2>
    • copy the file from another branch to current one

git log

  1. shows the current HEAD and its ancestry
    • prints the commit HEAD points to, then its parent, its parent and so on
    • it tranverse back through the repo’s ancestry, by revursively looking up each commit’s parent
  2. git log --oneline --graph --decorate

    git reflog

  3. an ordered list of the commits that HEAD has points to: it’s undo history for your repo

Remove files from git commit

  1. git reset – soft HEAD^/HEAD~1
  2. git reset HEAD
  3. git commit -c ORIG_HEAD

Delete branch

  1. Delete local branch
    git branch -d
    git branch -D => -D means force delete

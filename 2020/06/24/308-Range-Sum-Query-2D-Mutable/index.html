308. Range Sum Query 2D - Mutable

2020-06-24 Algorithms PV:

Solution I

  1. 建立列之和的矩阵： 就是(i, j)就是(0, j) + (1, j) + … + (i, j) 之和
  2. 相当于把多个一维的区间之和拼到了一起， 所以在构造函数中需要建立起这样一个列之和的矩阵， 然后在更新某一个位置时， 我们只需要将该列中改变的位置下面的所有数字更新一下即可
  3. 求某个区间的和时， 只要将相差的各列中对应的起始和结束的行上的差值累加起来即可。
